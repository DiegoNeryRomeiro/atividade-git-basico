# Atividade Prática: Fluxo de Trabalho Git/GitHub

Este repositório foi criado como parte de uma atividade acadêmica, da disciplina de engenharia de software para demonstrar o domínio das ferramentas de versionamento de código. O foco principal é a execução do fluxo básico de desenvolvimento utilizando Git e a plataforma GitHub.

## 🎯 Objetivos do Projeto
* Criar e configurar um repositório remoto.
* Clonar o repositório para o ambiente local.
* Realizar o versionamento de arquivos através de commits.
* Sincronizar as alterações locais com o servidor remoto (Push).

## 🛠️ Tecnologias Utilizadas
* **Git**: Sistema de controle de versão distribuído.
* **GitHub**: Plataforma de hospedagem de código.
* **VS Code**: Editor de código principal.

## 📝 Histórico de Commits
Neste projeto, foram registrados pelo menos 3 commits seguindo boas práticas de mensagens claras, simulando a evolução de um software real.
